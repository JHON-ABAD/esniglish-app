# 🌐 Esniglish Web App

[![Deploy to Vercel](https://img.shields.io/badge/Deployed_on-Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://esniglish-app.vercel.app/)

> El futuro del lenguaje es lógico. Plataforma educativa interactiva para aprender Esniglish, un idioma construido desde la eficiencia, el respeto y la herencia cultural (diseñado para ser hablado con la velocidad del inglés y la claridad del japonés).

## 🚀 Características Principales

* **Motor Gramatical Dinámico:** Generación de oraciones y traducciones en tiempo real (Sujeto + Verbo + Objeto) mediante constructores de estado.
* **Módulos de Práctica Interactiva:** Secciones dedicadas para conjugación de verbos, posesivos, adjetivos y estructuración de preguntas.
* **Diccionario Sincronizado:** Base de datos en la nube para gestionar vocabulario, partículas de tiempo y reglas fonéticas.
* **Autenticación y Sesiones:** Sistema de login seguro.
* **Interfaz Moderna y Responsiva:** Diseño minimalista enfocado en la experiencia de usuario (UX) y retención del aprendizaje.
